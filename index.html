In this project, a projectile motion computer program is built. With the input values and the calculated points, the program draws the projectile motion graph of an object and using the kinematic formulas, it finds the unknown parameters.

Projectile motion is a common motion in our daily lives. Golf balls in flight, a football kicked in a game, a cannonball fired from a cannon, a bullet fired from a gun are all examples of projectile motion. As a definition, projectile motion is the motion of an object projected into the air, which the only acting force is gravity. . In that case, the object is called a projectile, and the path it goes is called its trajectory.
